# Domeneshop CLI

Velkommen til Domeneshop CLI wiki!

## Hva er Domeneshop CLI?

Et kraftig kommandolinjeverktøy for å administrere dine Domeneshop-tjenester via deres [offisielle API](https://api.domeneshop.no/docs/).

## Funksjoner

- **Domener** - List og vis domenedetaljer
- **DNS** - Full CRUD for DNS-poster (A, AAAA, CNAME, MX, TXT, SRV)
- **Videresendinger** - Administrer HTTP-videresendinger
- **Fakturaer** - Oversikt over fakturaer
- **DDNS** - Dynamisk DNS-oppdatering

## Hurtigstart

### macOS
```bash
./domeneshop.command
```

### Windows
```batch
domeneshop.bat
```
