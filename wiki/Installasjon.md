# Installasjon

## Krav

- Python 3.8 eller nyere
- pip (Python package manager)

## Enkel installasjon (anbefalt)

### macOS
1. Last ned eller klon repositoriet
2. Dobbeltklikk på `domeneshop.command`
3. Virtuelt miljø og avhengigheter installeres automatisk

### Windows
1. Last ned eller klon repositoriet
2. Dobbeltklikk på `domeneshop.bat`
3. Virtuelt miljø og avhengigheter installeres automatisk

## Manuell installasjon

```bash
# Klon repositoriet
git clone https://github.com/OfficialLexthor/Domeneshop-CLI.git
cd Domeneshop-CLI

# Installer med pip
pip install -e .

# Eller installer avhengigheter manuelt
pip install -r requirements.txt
```

## Verifiser installasjon

```bash
domeneshop --version
domeneshop --help
```
